# PacketScope

A high-performance network packet analyzer built in C++ for packet capture, protocol inspection, and traffic analysis.

PacketScope provides visibility into network communication by capturing packets, decoding protocol headers, and analyzing traffic across multiple layers of the TCP/IP stack.

## Features

* Real-time packet capture and inspection
* Protocol header parsing and decoding
* TCP/IP traffic analysis
* Source and destination endpoint identification
* Packet statistics and monitoring
* Network troubleshooting and protocol visibility

## Tech Stack

* C++
* Socket Programming
* TCP/IP Networking
* Packet Capture
* Protocol Analysis

## Architecture

